Setup Guide

WordPress Setup

1. Create a new WordPress install.

2. Set permalink structure to Post name.

3. Install Hello Elementor.

4. Install Elementor and supporting plugins.

5. Create pages: Home, About, Services, Portfolio, Testimonials, Blog, Contact, Book Consultation.

6. Set Home as the static front page.

7. Create menu: Home, About, Services, Portfolio, Testimonials, Blog, Contact.

8. Add Book Consultation as the primary header CTA.

Elementor Global Settings

Colors:

  • Primary: #111111
  • Secondary: #F8F5F2
  • Accent: #C8A96B
  • Text: #111111
  • Muted: #D9D9D9

Fonts:

  • Primary Headings: Playfair Display
  • Body: Inter

Layout:

  • Content width: 1200px
  • Container padding mobile: 14px each side
  • Container padding desktop: 20px each side

CSS Installation

Copy wordpress-site-files/custom-css/lumora-global.css into one of these:

  • Elementor Site Settings > Custom CSS
  • Appearance > Customize > Additional CSS
  • Child theme stylesheet
